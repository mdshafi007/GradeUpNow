
# GradeUpNow

GradeUpNow is an educational platform that I and my friends are building together, aiming to create the best learning platform, offering a unique experience with better UI, valuable content, and smarter features than existing platforms.

GradeUpNow offers everything a B.Tech student needs— coding tutorials, computer science subjects, tech skills like LinkedIn and GitHub and lot more!


## Features

- Course Tutorials: Step-by-step tutorials on programming languages and computer science subjects.

- Notes: Students can write, save, and categorize their notes into various subjects.

- User Authentication: Students can register, log in, and track their progress.

- Notifications: Get notified about relevant internship and job opportunities.

- **AI-Powered Features (Planned)**:

  - Real-Time Doubt Solving: AI-powered chatbot for answering student queries in real-time.

   - AI-Generated Summaries: Topic summaries generated using AI for quick revision.

   - Course Recommendations: AI-driven suggestions for relevant courses based on user input.


## Tech Stack

**Frontend:** React.js, Bootstrap

**Backend:** Node.js, Express.js

**Database:** MongoDB

**AI Integration:** OpenAI GPT-4 for real-time doubt answering (planned)
