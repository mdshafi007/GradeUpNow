- **Heading:** `#1a1a1a` (Dark, high contrast)
- **Body:** `#666` (Medium gray, good readability)
- **Muted:** `#999` (Light gray, secondary text)

### Background:
- **Main:** `#ffffff` (Pure white)
- **Subtle:** `#f5f5f5` (Very light gray for dividers)

## 🎯 Key UI Components

### 1. Hero Badge
```css
Background: #FFF5EB (light orange)
Text: #FF7700 (orange)
Border: #FFE5CC (orange border)
```

### 2. Tabs
```css
Active: Orange bottom border + light orange background
Hover: Light orange background
Default: Gray text
```

### 3. Topic Cards
```css
Background: White
Border: #f0f0f0
Shadow: Subtle on default, orange tint on hover
Icon: Orange background (#FFF5EB)
Button: Orange (#FF7700)
```

### 4. Difficulty Badges
```css
Easy: Green (#10B981)
Medium: Amber (#F59E0B)
Hard: Red (#EF4444)
All with 15% opacity background
```

## 📱 Responsive Design

### Desktop (1200px+):
- 3 cards per row
- Full spacing
- All text visible

### Tablet (768px-1199px):
- 2 cards per row
- Reduced spacing
- All text visible

### Mobile (< 768px):
- 1 card per row
- Compact spacing
- Tab text hidden (icons only)
- Full-width buttons

## ✨ Interactions & Animations

### Card Hover:
- Lifts up 4px
- Orange shadow appears
- Border becomes orange-tinted
- Button arrow shifts right

### Button Hover:
- Darker orange background
- Subtle shadow
- Smooth transition (0.3s)

### Tab Click:
- Smooth active state transition
- Orange bottom border appears
- Background tints orange

## 💡 Usage Notes

### Adding New Topics:
- No need to specify `color` anymore
- All cards use consistent orange icon style
- Just add: title, description, difficulty, icon emoji
- Difficulty badge colors are automatic

### Customization:
- Change `#FF7700` to your brand color (search & replace)
- Adjust spacing in CSS if needed
- All colors are consistently named
