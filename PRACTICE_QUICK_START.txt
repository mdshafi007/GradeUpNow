➕ HOW TO ADD NEW TOPIC (30 seconds!)
───────────────────────────────────────────────────────────────────
Supabase → Table Editor → practice_topics → Insert row

Fill in:
  • category_id: [Select: Programming or Practice Tests]
  • title: "Computer Networks"
  • description: "OSI Model, TCP/IP, Routing"
  • difficulty: "MEDIUM"
  • icon: "🌐"
  • color: "#8B5CF6"
  • total_questions: 0
  • is_active: ✓ true
  • order_index: 4

Click Save → Refresh browser → New card appears! 🎉
